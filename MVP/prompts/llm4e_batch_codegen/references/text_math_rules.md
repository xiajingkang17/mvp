# 文本与公式规则

## 基本分工

1. 纯中文或自然语言说明使用 `Text(...)`
2. 纯数学表达式使用 `MathTex(...)`
3. 混合内容拆成 `Text(...)` 与 `MathTex(...)` 后用 `VGroup(...)` 组合

## 明确禁止

1. 禁止把中文直接放进 `Tex(...)`
2. 禁止把中文直接放进 `MathTex(...)`
3. 禁止把整段同时含中文和公式的内容整体写进 `Text(...)`

## 数学内容识别

以下内容默认视为数学内容，优先用 `MathTex(...)`：

1. 希腊字母：`μ`、`α`、`β`、`ω`
2. 下标与上标：`a_A`、`v_B`、`R_1`
3. 分式与根式
4. 力学和电磁学常见量：`f_s`、`W_F`、`Q`、`B_1`、`E`

## 标签规则

1. 原点标签一律用 `O`
2. 坐标里的数字零一律用 `0`
3. 点名与坐标优先拆开：
   `VGroup(Text("P"), MathTex("(0,h)"))`
4. 标签优先用 `next_to(...)` 或明确锚点，不要压主图
