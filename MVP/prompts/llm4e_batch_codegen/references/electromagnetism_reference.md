# 电学与电磁参考

这份参考用于：

1. 电学电路图
2. 电场 / 磁场图示
3. 带电粒子运动图

目标是让 `llm4e` 先把结构画清楚，再考虑更复杂的组件表达。

默认原则：

1. 坐标系、区域、粒子、轨迹、关键点要分层清楚
2. 文字说明不能压主图
3. 区域标签、方向箭头、关键点标签都应简洁稳定

## 一、电学电路

适用场景：

- 电源、灯泡、电阻、电容、开关、电表
- 简单串并联与实验电路

推荐做法：

- 电路结构先清楚，元件不要挤在一起
- 元件标签与元件本体分开处理
- 说明文字放在电路图外侧，不要压在线路上

可参考的组件源码目录：

- `prompts/llm4e_batch_codegen/references/components/electricity/`

其中包括：

- `battery.py`
- `bulb.py`
- `capacitor.py`
- `resistor.py`
- `switch.py`

## 二、电场与磁场

### 电场

适用场景：

- 匀强电场方向说明
- 区域场分布

推荐做法：

- 用统一方向的箭头阵列表达
- 箭头数量适中，避免喧宾夺主
- 场强标签与箭头阵列分开摆放

### 磁场

适用场景：

- 出纸面点阵
- 入纸面叉阵

推荐做法：

- 出纸面优先点阵
- 入纸面优先叉阵
- 叉号优先用基础图元手工组合，不依赖不稳定写法

不要：

- 不要使用 `Cross(point)`
- 不要使用 `Cross(size=...)`

## 三、带电粒子运动

适用场景：

- 粒子进入电场 / 磁场 / 复合场
- 轨迹、关键点、圆心、半径、速度方向

推荐做法：

- 粒子点优先 `Dot`
- 速度箭头、受力箭头独立绘制
- 轨迹必须可见，不要只剩起点终点
- 关键点标签优先贴近点，不要压住轨迹或坐标轴

## 四、坐标系与布局

推荐做法：

1. 物理题优先简洁坐标轴
2. 原点标签一律 `O`
3. 坐标中的数字零一律 `0`
4. 第一幕里，题干区和主图区必须分开
5. 每问开场时，当前问题卡优先放左上角
6. 推导幕中，主图区和公式区要明显分开

## 常见错误

1. 区域标签、点标签、轨迹标签挤在一起
2. 轨迹画出来但没有真正播放运动
3. 粒子、箭头、公式、标题全压在坐标轴中心附近
4. 中文和数学符号混写进同一个 `Text(...)` 或 `MathTex(...)`
