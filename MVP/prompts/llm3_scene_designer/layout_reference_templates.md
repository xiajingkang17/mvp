# 布局参考模板（仅参考，不是硬枚举）

以下三种是高优先参考布局。你可以基于它们自由变化与混合，不要求严格一模一样。

## 模板 A：左图右公式（常规首选）

- 主图在左，公式推导在右。
- 左侧主图宽度建议占画面 `55% ~ 60%`。
- 右侧用于公式推导、要点文字、结论。
- 避免主图横向铺满全屏，尤其不要让主图长度接近屏幕总宽度。

## 模板 B：长图下置（长轨道/长装置场景）

- 当主图天然是长条形、横向跨度很大时：
- 主图放在下方区域，建议高度约占画面 `30%`。
- 上方区域用于标题、公式推导、关键结论。
- 目标是避免“长图压缩公式区”或“图文互相覆盖”。

## 模板 C：左上标题 + 左下图 + 右区推导

- 左上放标题或本段目标。
- 左下放主图（实验示意、轨迹图、装置图）。
- 右侧区域集中放公式推导与关键步骤。
- 适合“讲解 + 推导并行”的分镜。

## 使用原则

- 以上仅作为构图参考，不限制创意发挥。
- 在自由发挥时，优先保持可读性：主图、公式、标题三者不重叠。
- 若参考模板与当前 scene 叙事目标冲突，以叙事清晰优先。
