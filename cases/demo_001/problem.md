# 高中力学难题：斜面—粗糙段—弹簧—弹性碰撞—追及

如图：一光滑斜面与水平面平滑连接，斜面倾角为 \(\theta=37^\circ\)，取  
\(\sin37^\circ=0.6,\ \cos37^\circ=0.8\)，重力加速度 \(g=10\,\text{m/s}^2\)。

斜面顶端点 \(A\) 到斜面底端连接点 \(B\) 的竖直高度为 \(h=2.5\,\text{m}\)。  
从 \(B\) 向右是一段粗糙水平面 \(BC\)，长度 \(L=4.0\,\text{m}\)，动摩擦因数 \(\mu=0.25\)。  
\(C\) 右侧连接一段**光滑**水平面，其右端固定一轻质弹簧，弹簧的自由端面恰位于点 \(C\)，弹簧劲度系数 \(k=300\,\text{N/m}\)，初始时弹簧**不压缩**。

质量为 \(m_1=1.0\,\text{kg}\) 的小物块 \(P\) 从 \(A\) 由静止释放，沿斜面下滑到 \(B\)，进入粗糙段 \(BC\) 继续向右运动。  
质量为 \(m_2=3.0\,\text{kg}\) 的小物块 \(Q\) 初始静止放在点 \(C\)，与弹簧自由端接触（弹簧未压缩）。  
当 \(P\) 到达 \(C\) 时与 \(Q\) 发生**完全弹性正碰**（碰撞时间极短，忽略碰撞过程中的摩擦与位移）。除 \(BC\) 段外，其余接触面均光滑。

---

## 问题

1. 求物块 \(P\) 到达点 \(C\) **碰前瞬间**的速度大小 \(v_{P0}\)。

2. 求第一次碰撞后瞬间，\(P\) 与 \(Q\) 的速度大小与方向。

3. 碰后 \(Q\) 进入右侧光滑弹簧区并压缩弹簧。求弹簧的最大压缩量 \(x_{\max}\)。

4. 求 \(Q\) 从“刚碰后位于 \(C\)、开始压缩弹簧”到“再次回到 \(C\)、准备进入粗糙段 \(BC\)”所用的时间 \(\Delta t\)。

5. 在 \(Q\) 回到 \(C\) 并向左进入粗糙段后，判断它是否会追上（或追到）已经在粗糙段上运动的 \(P\)。  
   - 若会，求它们“相遇/接触”发生在距 \(C\) 多远处，并求相遇瞬间两者的速度；  
   - 若不会，说明理由。

6. 最终两个物块都会在粗糙段 \(BC\) 上停下。求它们各自最终停止位置到点 \(B\) 的距离（用“离 \(B\) 多远”表示）。

> 说明：计算过程中可用能量守恒、动量守恒、简谐运动关系；粗糙段上摩擦力大小恒为 \(f=\mu mg\)，方向总与运动方向相反。
